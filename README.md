# Kafka-C

`git clone https://github.com/denovo-mani/Kafka-C.git Kafka-C`

## Kafka Library for Ubuntu 
`sudo apt-get install librdkafka-dev`

## Pull The Latest
`git pull origin master`

## Compile
`gcc producer.c -o producer -lrdkafka -lm`

## Run Producer
`./producer`
